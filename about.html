About Us

Red Wagon is the leading Managed File Transfer software on the market.

Features

  • Web-based Software - Deploy the software on a hardened server on your network. Users access all features from any major browser.
  • Multiple Transfer Protocols - FTP, SFTP, FTPS, CIFS/SMB (Microsoft Network).
  • Secure Transfer - Securely transfer files over public and private neworks using encrypted file transfer protocols.
  • Secure Storage - Securely store files using multiple data encryption methods.
  • Automated File Transfers - Scheduled file transfer processes between trading partners, internal network locations.
  • Authenticated Users - Authenticates users against existing user repositories such as LDAP and Active Directory.
  • Custom Intermediate Processes - Execute custom file process prior to delivery.
  • History - Detailed history of every transfer.
  • Auditing - Detailed log of all changes made by users.

Transferring file(s) is as easy as 1-2-3


1) Create a new location for source

Users Define a "location" which is the source of the files to be transferred. This source can be a network drive, a server, SFTP (SSH) server, FTP(s) server.

2) Create a new location for destination

Similar to above step, this is for the destination.

3) Create an "integration"

Users define an "integration" which transfers files from your source location to your destination location. It is here where you define a schedule, whether it should be encrypted, etc.
